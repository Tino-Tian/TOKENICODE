关于安装包的"数字签名"
========================

🤔 什么是数字签名？
数字签名相当于软件的"身份证"——证明这个软件确实是我们发布的，中途没有被篡改。

⚠️ 当前状态：
NOVA Windows 版本的安装包暂未购买数字签名证书（EV Code Signing Certificate）。
这意味着：
- Windows 会弹出 SmartScreen 警告"Windows 保护了你的电脑"
- 这是正常的，不是你的电脑中了病毒
- 你只需要点"更多信息"→"仍要运行"即可继续安装

🔒 如何确认文件没被篡改？
1. 从 GitHub 官方发布页面获取 SHA256 校验值：https://github.com/yiliqi78/TOKENICODE/releases
2. 在电脑上按 Win+R，输入 powershell，回车
3. 输入命令（把文件路径换成实际的）：Get-FileHash "文件路径" -Algorithm SHA256
4. 对比显示的 hash 和 GitHub 页面上的 hash 是否一致
5. 如果一致 → 文件完整无误
6. 如果不一致 → 文件可能损坏或被篡改，请删除并重新下载

⚠️ 需要说明的是：
SHA256 校验只能证明"文件没被改过"，不能证明"文件一定是我们发布的"。
如果你对安全性有极高要求（如企业环境），请等待后续的签名版本。
