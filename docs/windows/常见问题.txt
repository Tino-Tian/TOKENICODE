NOVA Windows 版本 — 常见问题
================================

Q1: 安装时弹出"Windows 保护了你的电脑"，怎么办？
A1: 这是正常的。点"更多信息"，再点"仍要运行"就行。详见 U 盘中的"签名说明.txt"。

Q2: 安装后打开是白屏，什么都没有？
A2: 可能是缺少 WebView2。运行 U 盘 tools 文件夹里的 install-webview2-offline.cmd 试试。

Q3: 需要管理员权限才能安装吗？
A3: 不需要。NOVA 安装在你的个人文件夹里，不需要管理员权限。

Q4: 支持哪些 Windows 版本？
A4: Windows 10（1809 及以上）和 Windows 11。Windows 7/8 不支持。

Q5: 杀毒软件（360/火绒/腾讯管家/Defender）报毒怎么办？
A5: 这是误报。NOVA 没有签名证书所以容易被误判。请在杀毒软件中将 NOVA 安装目录添加为信任/白名单。

Q6: 怎么更新到新版本？
A6: NOVA 内置了自动更新功能。打开 NOVA 后，在设置中可以手动检查更新。

Q7: 安装包多大？
A7: 约 130-180MB（包含了 WebView2 离线安装组件）。

Q8: 能把 NOVA 装在 U 盘上直接用吗？
A8: 当前版本需要安装到电脑上运行。U 盘便携版计划在后续版本中提供。

Q9: 安装后桌面没有快捷方式？
A9: 安装时请勾选"创建桌面快捷方式"。如果忘记了，去开始菜单找 NOVA 文件夹，右键 → 固定到开始屏幕。

Q10: NOVA 和 Claude Code 是什么关系？
A10: NOVA 是 Claude Code 的桌面外壳，让你可以像用普通软件一样使用 Claude Code，不需要打开命令行。

Q11: 需要联网吗？
A11: 是的。Claude Code 的 AI 功能需要联网才能使用。

Q12: 需要自己安装 Claude Code 吗？
A12: 不需要。首次启动 NOVA 时会自动引导你安装所需的组件。

Q13: 怎么卸载？
A13: 开始菜单 → NOVA 文件夹 → 卸载 NOVA。或通过 Windows 设置 → 应用 → 卸载。

Q14: Mac 版和 Windows 版功能一样吗？
A14: 核心功能一样。Windows 版本是 Beta 版，某些细节可能有差异。

Q15: 遇到其他问题怎么办？
A15: 请运行 U 盘 tools 中的 diagnostics.ps1（右键 → 用 PowerShell 运行），把输出结果发给开发者。
