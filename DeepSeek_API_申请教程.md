# DeepSeek API Key 申请教程

## 三步搞定

**第一步**：打开 [deepseek.com](https://deepseek.com) → 点击右上角「注册」

**第二步**：登录后 → 左侧菜单「API Keys」→ 点击「创建 API Key」

**第三步**：复制生成的 Key → 打开 NOVA 设置 → 粘贴 → 完成

---

## 费用说明

注册即送 **500 万 Tokens** 免费额度。
后续按量付费，约 **¥1 / 百万 Tokens**，非常便宜。

---

## 注意事项

- API Key 不要发给任何人，泄露后别人可以花你的钱
- 如果 Key 泄露了，在 DeepSeek 后台删掉重新创建一个就行

---

## 安装后如何更换

1. 打开 NOVA
2. 点击设置 → 提供商管理
3. 选择「添加提供商」→ 选择「DeepSeek」
4. 粘贴你的 API Key
5. 点击保存 → 设为活跃

搞定。三分钟的事。
